Returns:

PyLegendBoolean

timedelta

PyLegendDate.timedelta(number, duration_unit)[source]

Shift the date by a given amount.

Parameters:

  • number (Union[int, PyLegendInteger]) – How many units to shift (negative values shift backwards).

  • duration_unit (str) – One of 'YEARS', 'MONTHS', 'WEEKS', 'DAYS', 'HOURS', 'MINUTES', 'SECONDS', 'MILLISECONDS', 'MICROSECONDS', 'NANOSECONDS' (case-insensitive).

Returns:

The shifted date.

Return type:

PyLegendDate

Raises:

  • TypeError – If number is not an int or PyLegendInteger.

  • ValueError – If duration_unit is not a recognised unit.

Examples

Download Interactive Notebook

import pylegend
frame = pylegend.samples.pandas_api.northwind_orders_frame()

frame["plus_7d"] = frame["Shipped Date"].timedelta(7, "DAYS")
frame[["Shipped Date", "plus_7d"]].head(3).to_pandas()
